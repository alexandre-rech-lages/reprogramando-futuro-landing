# Web App - Reprogramando Futuro

Aplicação web para o projeto Reprogramando Futuro.

## 📋 Descrição

Este é o frontend da aplicação Reprogramando Futuro, desenvolvido com tecnologias web modernas.

## 🚀 Tecnologias

- React
- Node.js
- JavaScript/TypeScript
- CSS/SCSS

## 📦 Instalação

```bash
npm install
```

## 🏃 Como Executar

```bash
npm start
```

A aplicação estará disponível em `http://localhost:3000`

## 🔨 Build para Produção

```bash
npm run build
```

## 📁 Estrutura do Projeto

```
src/
├── components/    # Componentes reutilizáveis
├── pages/         # Páginas da aplicação
├── styles/        # Estilos globais
└── utils/         # Funções utilitárias
```

## 📝 Scripts Disponíveis

- `npm start` - Inicia o servidor de desenvolvimento
- `npm run build` - Cria build otimizado para produção
- `npm test` - Executa testes
- `npm run eject` - Expõe configurações do webpack (irreversível)
